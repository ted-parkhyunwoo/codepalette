JavaScript 기본 컨테이너(컬렉션) 정리

✅ 1. Array (배열)

📌 선언
let arr = [1, 2, 3];
let empty = [];
➕ 요소 추가
arr.push(4);        // 뒤에 추가
arr.unshift(0);     // 앞에 추가
arr.splice(2, 0, 99); // 인덱스 2 위치에 삽입
➖ 요소 삭제
arr.pop();           // 뒤 삭제
arr.shift();         // 앞 삭제
arr.splice(2, 1);     // 인덱스 2 요소 삭제
🔀 정렬
arr.sort((a, b) => a - b); // 오름차순
arr.sort((a, b) => b - a); // 내림차순
🎯 요소 접근
console.log(arr[1]);
for (let i = 0; i < arr.length; i++) console.log(arr[i]);
for (let val of arr) console.log(val);

✅ 2. Object (객체)

📌 선언
let obj = { name: "Ted", age: 30 };
➕ 요소 추가/수정
obj.city = "Seoul";
obj["age"] = 31;
➖ 요소 삭제
delete obj.age;
🔀 정렬
// 객체 배열 정렬 예시
let users = [
  { name: "Ted", age: 30 },
  { name: "Amy", age: 25 }
];
users.sort((a, b) => a.age - b.age);
🎯 요소 접근
console.log(obj.name);
console.log(obj["age"]);
for (let key in obj) console.log(key, obj[key]);
Object.keys(obj);
Object.values(obj);
Object.entries(obj);

✅ 3. Set

📌 선언
let set = new Set([1, 2, 3]);
➕ 요소 추가
set.add(4);
➖ 요소 삭제
set.delete(2);
set.clear();
🔀 정렬
let sorted = [...set].sort((a, b) => a - b);
🎯 요소 접근
for (let val of set) console.log(val);
let arr = [...set];
console.log(arr[0]);

✅ 4. Map

📌 선언
let map = new Map();
map.set("name", "Ted");
➕ 요소 추가/수정
map.set("age", 30);
map.set("age", 31);
➖ 요소 삭제
map.delete("age");
map.clear();
🔀 정렬
let sortedEntries = [...map.entries()].sort((a, b) => a[1] - b[1]);
🎯 요소 접근
map.get("name");
for (let [key, val] of map) console.log(key, val);
map.keys();
map.values();
map.entries();

📋 요약 테이블

컨테이너 선언 예시 추가 삭제 정렬 접근 방식 중복 허용 여부
Array [1,2,3] push/unshift/splice pop/shift/splice sort() 인덱스 (arr[i]) O
Object {a:1} obj.key = val delete obj.key 불가 (객체 배열만 가능) 키 기반 (obj[key]) X (키 중복 불가)
Set new Set() add() delete(), clear() [...set].sort() 값 반복 (for...of) X
Map new Map() set() delete(), clear() [...map].sort() 키 기반 (get/set) X (키 중복 불가)
